Jeeva SHG Manager - Iframe Scrolling Test

This page tests the iframe scrolling functionality of the Jeeva SHG Manager app.

Testing Instructions:

  • Try scrolling within the iframe using mouse wheel, touch gestures, or scrollbar
  • Navigate to different screens within the app (Dashboard, Loans, Payments, etc.)
  • Check if the content scrolls properly on both desktop and mobile
  • Verify that the iframe doesn't interfere with the parent page scrolling

Iframe Attributes Used:

  • scrolling="auto": Enables scrolling within the iframe
  • allowfullscreen: Allows the iframe to go fullscreen
  • sandbox: Security restrictions while allowing necessary functionality
  • height="80vh": Sets iframe height to 80% of viewport height
